# DayQ

اپ آفلاین مدیریت کار روزانه — برای مدرپ‌های میدانی دارویی و دستیار مارکتینگ.
تک‌فایل HTML، فارسی RTL، تقویم شمسی، بدون نیاز به سرور یا حساب کاربری.

## استفاده

از این لینک باز کن:
**https://boojimmd.github.io/Dayq/**

روی آیفون: با Safari باز کن → دکمهٔ Share → Add to Home Screen
روی اندروید/ویندوز: همان لحظهٔ باز‌کردن، Chrome خودش گزینهٔ نصب را نشان می‌دهد

بعد از نصب، کاملاً آفلاین کار می‌کند — دیگر نیازی به اینترنت نیست.

## ویژگی‌ها

- ثبت سریع کار با تشخیص هوشمند تاریخ/ساعت/اولویت از روی متن فارسی
- تقویم شمسی (ماهانه/هفتگی/فهرست)
- سینک چنددستگاهی با یک کد ۶رقمی
- یادآوری Push
- پروژه‌ها، یادداشت روزانه، آرشیو خودکار

## تست

قبل از هر تغییر جدید:
```
npm install playwright
node test_dayq.js
node test_sync.js
node test_ics.js
```
اگر چیزی FAIL داد، یعنی یک فیچر قبلی جایی خراب شده — دست به کد جدید نزن تا اول همین را پیدا کنی.
