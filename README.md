# DayQ

اپ آفلاین مدیریت کار روزانه — برای مدرپ‌های میدانی و دستیار مارکتینگ.
تک‌فایل HTML، فارسی RTL، بدون نیاز به سرور.

## استفاده
فقط `DayQ.html` را در مرورگر باز کن، یا در آیفون با Safari باز کن و Add to Home Screen بزن.

## تست
قبل از هر تغییر جدید:
```
npm install playwright
node test_dayq.js
node test_sync.js
```
اگر چیزی FAIL داد، یعنی یک فیچر قبلی جایی خراب شده — دست به کد جدید نزن تا اول همین را پیدا کنی.
